#### 在医院实习


我们本科第一年在北大校园里，当时住在一个新建的宿舍楼里感觉很开心

第二年第三年就搬到医学部的校园了。

然后第四年第五年就搬去医院了。我入学的时候是临床3班，我们一共有5个班，每个班对应着不同的实习医院，我们班对应的就是积水潭医院，在北师大附近。于我们就搬到附近的胡同里去住了

就在那一年，积水潭医院在回龙观建好了一个新园区，大部分科室都搬过去了。。。于是我们就得坐班车从西三环去回龙观（早上6点爬起来赶班车），提前体验了一下社畜的生活。

我们班所有的人都在这里了，因为定向的是骨科医院，所以女生比较少（男女2:1）。二班定向的是北京医院，是一个老干部医院，所以女生比较多 （杨一峰的班）。我们两个班就有种难兄难弟的感觉（因为这两个医院都不是特别好的综合医院）。

中间的是带教的陆老师，她人超级超级认真。

#### 2014年 igem


本科的时候也干了一堆乱七八糟的事情：学了个双学位；还做了社团（本部的社团），一个开源软件协会的副会长；在医学部团学联的网路部做副部长（给人剪视频的那种，肝）；在一个生物物理实验室搬砖（还去你们人大做过实验）；还有就是这个igem。

这个就是我在本科忙到吐血的东东了。

来自百度百科：国际遗传工程机器设计竞赛（英语：International Genetically Engineered Machine Competition, iGEM）是一年一度的世界级的合成生物学竞赛。该竞赛由麻省理工学院于2004年首创，旨在培养合成生物学人才，促进各国大学本科生在该领域的学习、交流与合作。每年，大赛主办方提供给各参赛队伍一份DNA标准样本库，各队自行选题并通过设计和模型分析，将所需样本导入现有的生物体系，合成全新的生物工程系统（即“基因工程机器”）。

每年pku都会组队参加，完全由学生组织，没有知道老师，都是学长学姐带新人这个样子。

需要我们自己设计项目，自己运营实验室。

这个项目持续了1年多，开始的半年是选拔阶段，最后还考了一天试才确定最后的队员。14年11月来波士顿比了赛。

大家每天都熬到很晚，一起为实验焦虑啥的，所以还有挺深厚的“革命友谊”的。我打疫苗去世的那个同学就是我上一届参加这个活动的队员。

我们的advisor也是生科的同学，一个叫郑璞，一个叫刘时煜（我后来duke的男舍友）。

他们两个脾气都比较暴躁233，特别是郑璞。经常我有问题要问他的时候，他就回我 “我很烦，去找刘时煜”

因为“同行衬托”，所以显得刘老师脾气还好一点（但其实不是，大意了😂）

我跟ex就是这里认识的。他脾气也很暴躁，他们三个因为最后发paper作者排序的问题吵了好久。。。。
