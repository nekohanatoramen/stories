#### 在医院实习


我们本科第一年在北大校园里，当时住在一个新建的宿舍楼里感觉很开心

第二年第三年就搬到医学部的校园了。

然后第四年第五年就搬去医院了。我入学的时候是临床3班，我们一共有5个班，每个班对应着不同的实习医院，我们班对应的就是积水潭医院，在北师大附近。于我们就搬到附近的胡同里去住了

就在那一年，积水潭医院在回龙观建好了一个新园区，大部分科室都搬过去了。。。于是我们就得坐班车从西三环去回龙观（早上6点爬起来赶班车），提前体验了一下社畜的生活。

我们班所有的人都在这里了，因为定向的是骨科医院，所以女生比较少（男女2:1）。二班定向的是北京医院，是一个老干部医院，所以女生比较多 （杨一峰的班）。我们两个班就有种难兄难弟的感觉（因为这两个医院都不是特别好的综合医院）。

中间的是带教的陆老师，她人超级超级认真。

#### 舍友

这个是13年六一儿童节我们宿舍拍的，大家集体扎了两个辫子。

左边是我，张一帆（阿帆，我唐山老乡），梁静汝（豆豆，我们班第一年的团支书，兰州人），苗荷佳（阿喵，西安人）

刚刚上大学的时候就立志要好好学习哈哈哈

莫名其妙产生了一种执念：我要坚持坐第一排😂 

于是成了我们宿舍每天起最早，回来最晚的人lol

大概每天6:30起床，洗漱，去教室占座

7点去操场打卡跑圈，7:30在食堂买个面包和咖啡（那时候就咖啡上瘾了）

然后看会书等8点上课

晚上下课回宿舍洗个澡，然后出门自习到图书馆关门（自律lol）

我们在宿舍床上装了床帘，有点隐私空间。阿帆判断我有没有起床就看我的鞋子还在不在。有一次她看见我的鞋还在外面，就想 “秋 还没起床，我就在睡会吧”，于是那天她就迟到了😂 我换了双鞋粗门了，没把旧的收起来233
